## Resumo do Tratamento de Dados

Tratamos dados pessoais apenas quando necessário para o funcionamento do nosso site, ou com base no seu consentimento ou interesse legítimo.

### Arquivos de Log do Servidor
Ao visitar nosso site, dados técnicos (endereço IP, navegador, data/hora) são coletados automaticamente para segurança e estabilidade.
- **Base legal:** Art. 6(1)(f) RGPD (Interesse legítimo).

### Formulários de Contato
Se você usar nossos formulários de contato, tratamos os dados que você fornece (nome, email, mensagem) para gerenciar sua solicitação.
- **Base legal:** Art. 6(1)(b) RGPD (Contrato/Pré-contratual) ou Art. 6(1)(a) (Consentimento).

### Cookies e Análise (Google Analytics 4)
Usamos o Google Analytics 4 para analisar o uso do site. Este serviço usa cookies e processa dados (incluindo IPs truncados) para nos fornecer estatísticas anônimas.
- **Base legal:** Art. 6(1)(a) RGPD (Consentimento). Você pode revogar isso através das nossas Configurações de Cookies.

### Mídia Externa (YouTube)
Incorporamos vídeos do YouTube. A conexão com os servidores do YouTube é estabelecida apenas se você consentir com "Mídia".
- **Base legal:** Art. 6(1)(a) RGPD (Consentimento).

### OpenStreetMap
Usamos o OpenStreetMap para mostrar mapas. Isso é necessário para mostrar nossa localização. Seu endereço IP pode ser transmitido para a Fundação OpenStreetMap para este fim.
- **Base legal:** Art. 6(1)(f) RGPD (Interesse legítimo).
